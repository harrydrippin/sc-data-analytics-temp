## Git 작업 규칙

- **commit, push, branch 생성, PR 생성은 반드시 사용자 확인 후** 진행한다. 프로젝트 마무리 단계에서 물어본다.
- **merge는 반드시 사용자 확인 후** 진행한다.
- **PR 생성 시 반드시 새 브랜치를 생성**한다. 현재 작업 브랜치에서 직접 PR을 올리지 않고, 목적에 맞는 이름의 새 브랜치를 만들어 push한다.
